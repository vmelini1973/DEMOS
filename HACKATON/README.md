# Aprendizaje de desarrollo web – Prácticas de código

## Objetivo de este repositorio

Este repositorio es mi espacio de práctica y aprendizaje en desarrollo web.  
Aquí iré guardando **todos los ejercicios** que vaya realizando a lo largo de mi proceso de formación, desde lo más básico hasta tecnologías más avanzadas.

El objetivo es:

- Practicar de forma constante y progresiva.
- Acumular un historial de ejercicios que refleje mi evolución.
- Poder revisar, corregir y mejorar mi código con el tiempo.
- Construir un portfolio básico de prácticas de código.

## Tecnologías que voy a practicar

En este repositorio iré trabajando con:

- **HTML**: estructura básica de páginas web, etiquetas semánticas, formularios, etc.
- **CSS**: estilos, layout, centrado, colores, tipografía, flexbox, grid, etc.
- **JavaScript**: interactividad, manipulación del DOM, eventos, funciones, etc.
- **PHP**: backend básico, formularios, conexión a datos, etc.

## Flujo de trabajo

1. Escribo el código en Visual Studio Code.
2. Lo organizo en la carpeta correspondiente (HTML, CSS, JS, PHP).
3. Lo subo a este repositorio con un mensaje descriptivo.
4. Recibo correcciones y sugerencias.
5. Aplico las mejoras y actualizo el código.
